# Book API — REST-сервис на Go

Простой, но современный HTTP-сервер для управления коллекцией книг. Реализует полный CRUD, потокобезопасное in-memory хранилище и **graceful shutdown**. Отличный стартовый проект для понимания бэкенд-разработки на Go.

## Возможности

- **CRUD операции** над книгами:
  - `POST /books` – добавить книгу
  - `GET /books` – получить все книги
  - `GET /books/{id}` – получить книгу по ID
  - `PUT /books/{id}` – обновить книгу
  - `DELETE /books/{id}` – удалить книгу
- Потокобезопасное хранение в памяти (`sync.RWMutex`)
- Graceful shutdown при получении сигналов `SIGINT`/`SIGTERM`
- Логирование запросов
- Валидация входных данных
- Разделение на слои: модели, хранилище, обработчики
- Таймауты сервера (чтение, запись, idle)

## Технологии

- Go 1.21+
- Только стандартная библиотека (`net/http`, `encoding/json`, `sync`, `context`, `os/signal`)
